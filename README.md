# E-Commerce Full Stack

## Descripción
Proyecto de comercio electrónico completo con **panel administrativo** y **tienda para clientes**. Permite gestionar productos, pedidos y usuarios desde el backend, mientras que los clientes pueden navegar el catálogo, agregar productos al carrito y realizar compras de manera segura.

## Características Principales
- Gestión de productos (CRUD) y categorías para administradores.
- Autenticación y autorización de usuarios (JWT + roles admin/customer).
- Carrito de compras persistente y proceso de checkout.
- Gestión de pedidos y seguimiento de estado.
- Interfaz responsive y moderna tanto para administrador como para cliente.

## Tecnologías Utilizadas

| Categoría | Tecnologías |
|-----------|-------------|
| **Frontend** | React 18+, Vite, React Router v6, Axios, React Context + useReducer, React Hook Form, CSS Modules, React Icons, React Toastify |
| **Backend** | Node.js 20 LTS, Express.js, Sequelize, JWT, bcryptjs, Multer, Express Validator, dotenv, CORS |
| **Base de Datos** | MySQL 8 (desarrollo), MySQL AWS RDS (producción) |
| **Hosting / Infraestructura** | Comming soon |

## Instalación y Ejecución Local
1. Clonar repositorio:  
   ```bash
   git clone https://github.com/LeticiaTejeroGamboa2401/E-Commerce.git
2. Configurar variables de entorno en .env tanto en frontend como en backend.
3. Instalar dependencias:
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
4. Ejecutar en modo desarrollo:
   ```bash
   cd backend && npm run dev
   cd ../frontend && npm run dev
5. Abrir la aplicación en el navegador (generalmente http://localhost:5173).
